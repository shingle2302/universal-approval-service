## **三、关键路径（必须盯紧）**

1. **模板部署闭环（W3\~W4）**：不稳定会拖垮后续流程/任务开发
2. **可靠消息（W7\~W8）**：不上线前必须完成
3. **复杂场景（W9\~W10）**：建议独立质量门禁
4. **联调与压测（W12）**：至少预留 1 周，不可压缩

## **四、每周验收门禁（DoD）**

- 代码：单元测试覆盖关键逻辑（建议核心域 > 70%）
- API：OpenAPI 文档实时同步
- 数据：DDL/索引与查询计划评审通过
- 可观测：新增功能必须有日志、指标、trace
- 稳定性：失败重试/幂等可验证
- 安全：新增接口默认受鉴权保护
